## IAT Study

Questo repository contiene un esperimento basato su **Minno.js** che esegue un test di associazione implicita (IAT) seguito da un breve questionario socio-demografico.

## Descrizione del Progetto
Il progetto mira a indagare le associazioni automatiche tra le categorie Paid Media e Organic e i concetti Performance e Ostacoli. 
- **Struttura:** 1. Consenso Informato.
  2. Istruzioni IAT.
  3. Test IAT.
  4. Questionario Socio-Demografico.
  5. Pagina di ringraziamento.

## Tecnologie utilizzate
- [Minno.js](https://minnojs.github.io/): Framework principale per la creazione di test psicologici web-based.
- JavaScript: Utilizzato per la logica di presentazione dei test.
- GitHub Pages: Per l'hosting del progetto.

## Come eseguire il progetto

## Struttura dei file
- `index.html`: File di avvio del test.
- `mgr.js`: Gestisce la sequenza dei task.
- `iat.js`: Configurazione specifica per il test IAT.
- `socio_demografia.js`: Logica del questionario finale.
- `consent.js` / `ringraziamento.js`: Pagine di inizio e fine.
